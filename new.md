MarkDown的使用方法 
=  

## 关于文本

**1. 文本换行：**  
①先打两个空格，再按Enter 
②换段落：连续打两个Enter(起到两者之间有一空白行的作用)  

**2. 文本加粗：**  
前后分别打2个*或两个_

**3. 斜体：**  
首位分别加一个*或_  
如*你好*  
_Hello_  

**4. 既斜体又加粗**  
三个*   
如***Beauty***  
**5.高亮**  
==这是高亮==

## 关于段落  

**1. 加中划线/波浪线：**  
前后打两个~  
如~~我是废物~~  
**2. 加分割线：**  
在一空白行加三个及以上的*或_或者三个-（中间必须有空格）  

## 关于标题  
#后面紧跟一个空格（几个#就是几级标题）  

还可以在标题后一个空白行打=表示一级  
打-表示二级，但是之后需要再添加一个空白行  

## 关于列表  
**1. 无序列表：**  
*或+或-后跟空格（但要表示同一个列表就必须统一符号），如
* 我喜欢金泰亨  
* 金泰亨是世首帅  
+ KPOP很好听  

**2. 有序列表：**  
直接1.后加一个空格（第一项可以自定义，但是后面都要递增），如  
1. 微积分  
2. 线性代数  

**3. 列表的嵌套：**  
二级列表需要在*前面加4个空格，三级8个，如  
* 国漫  
    * 沧元图  
    * 仙逆  
    * 斩神  
        * 林七夜  
        * 沈青竹  
* 日漫  
    1.  柯南  
    2. 文豪野犬  
    3. 鬼灭之刃  
        1. 炭治郎  
        2. 富冈义勇

**4. 结束列表，重回段落：**  
按2次Enter  

**5. 勾选框列表：**  
*空格[空格]空格（注意[]必须是英文输入法），如

- [ ] 学习打字  
- [ ] 学习C语言  
- [x] 按时睡觉（x大小写都可以）  

## 关于代码块  
1. 在代码前面加四个空格（每一行都加，直接全选再按Tab键）  
2. 用```行开始，之后以同样方式结束，如下面是一段代码  
```c  
   #include  <stdio.h>

    int main()
    {
    printf("Hello!\n");
    return 0;

    }  
```  
3. 行内代码块  
    我要调用这段代码`
    printf("Hello!\n");`
    
## 关于引用  
    1. 直接引用  
    用>加空格（退出引用需隔一行）(示例中有换行换段方法)
    > 《红楼梦》被评为中国古典章回小说的巅峰之作，思想价值和艺术价值极高。
    > 
    > 原本前80回尚存。  
    > 以贾宝玉、林黛玉、薛宝钗等人物的爱情、婚姻与命运

     最好每一行前面都加>,便于读源代码并且与列表，代码块，引用等嵌套  
     > 红楼梦人物  
     > * 贾宝玉  
     > * 薛宝钗  
## 关于超链接  
1. 把需要超链接的数字用[]括起来，后面紧跟（网址），如  
[油管](https://www.youtube.com/)  
2. 如果一段话里需要插入很多超链接，还有一种较为直观的方法，示例如下  
[克劳德][a]是游戏[最终幻想][b]里的男主角  

[a]:https://zh.wikipedia.org/w/index.php?title=%E5%85%8B%E5%8B%9E%E5%BE%B7%C2%B7%E5%8F%B2%E7%89%B9%E8%90%8A%E5%A4%AB&oldformat=true&variant=zh-cn  
[b]:https://zh.wikipedia.org/wiki/%E6%9C%80%E7%BB%88%E5%B9%BB%E6%83%B3V
